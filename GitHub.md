# GitHub

## Sync with Remote from fork

> [Windows, Linux & Mac]
>```
> git remote add FNLHub https://github.com/FNLHub/Git-Intro.git
>```
>```
> git pull --rebase FNLHub main
>```
> If the conflicts are not true, skip the patches.
>```
> git rebase --skip
>```
>```
> git push --force-with-lease origin main
>```
